List ADT

Instantiation:

List<type> listName = new List<type>();

Methods:

listName.add(x); Adds element x to the end of the list
listName.add(x,y); Adds element y to the list at index x
listName.contains(x); Returns a boolean true if the list contains element x, false if it does not
listName.get(x); Returns the element at index x
listName.indexOf(x); Returns the index of element x
listName.isEmpty(); Returns a boolean true if the list is empty, false if the list contains at least one element
listName.populate(x); Empties the list, then adds x randomly generated elements of appropriate type to the list
listName.size(); Returns the size of the list
listName.remove(x); Removes the element at index x in the list
listName.set(x,y); Replaces the element at index x in the list with element y
listName.clear(); Empties the list

Stack ADT

Instantiation:

Stack<type> stackName = new Stack<type>();

Methods:

stackName.peek(); Returns the top item in the stack, but does not remove it
stackName.push(x); Adds element x to the top of the stack
stackName.pop(); Removes and returns the top item in the stack
stackName.isEmpty(); Returns a boolean true if the stack is empty, false if the stack contains at least one element
stackName.populate(x); Empties the stack, then adds x randomly generated elements of appropriate type to the stack
stackName.search(x); Returns the index of value x in the stack, where an index of 1 represents the top of the stack

Queue ADT

Instantiation:

Queue<type> queueName = new Queue<type>();
PriorityQueue<type> queueName = new PriorityQueue<type>();

Methods:

queueName.peek(); Returns the first item in the queue, but does not remove it
queueName.add(x); Adds element x to the end of the queue
queueName.remove(); Removes and returns the first item in the queue
queueName.isEmpty(); Returns a boolean true if the queue is empty, false if the queue contains at least one element
queueName.populate(x); Empties the queue, then adds x randomly generated elements of appropriate type to the queue
queueName.size(x); Returns the size of the queue
